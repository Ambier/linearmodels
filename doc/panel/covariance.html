Panel Model Covariance Estimators¶

class HomoskedasticCovariance(y, x, params, df_resid)[source]¶

Homoskedastic covariance estimation

Parameters:
  • y (ndarray) – (entity x time) by 1 stacked array of dependent
  • x (ndarray) – (entity x time) by variables stacked array of exogenous
  • params (ndarray) – variables by 1 array of estimated model parameters
  • df_resid (int) – Residual degree of freedom to use when normalizing covariance

Notes

The estimator of the covariance is

\[s^2\hat{\Sigma}_{xx}^{-1}\]

where

\[\hat{\Sigma}_{xx} = (NT)^{-1}\sum_{i=1}^N\sum_{t=1}^T x_{it}x_{it}^{\prime}\]

and

\[s^2 = (NT)^{-1}\sum_{i=1}^N\sum_{t=1}^T \hat{\epsilon}_{it}^2\]

where NT is replace by NT-k if debiased is True.

class HeteroskedasticCovariance(y, x, params, df_resid)[source]¶

Covariance estimation using White estimator

Parameters:
  • y (ndarray) – (entity x time) by 1 stacked array of dependent
  • x (ndarray) – (entity x time) by variables stacked array of exogenous
  • params (ndarray) – variables by 1 array of estimated model parameters
  • df_resid (int) – Residual degree of freedom to use when normalizing covariance

Notes

The estimator of the covariance is

\[\hat{\Sigma}_{xx}^{-1}\widehat{Cov}(x_{it}\epsilon_{it})\hat{\Sigma}_{xx}^{-1}\]

where

\[\hat{\Sigma}_{xx} = (NT)^{-1}\sum_{i=1}^N\sum_{t=1}^T x_{it}x_{it}^{\prime}\]

and

\[\widehat{Cov}(x_it\epsilon_{it}) = (NT)^{-1}\sum_{i=1}^N\sum_{t=1}^T \hat{\epsilon}_{it}^2 x_{it}x_{it}^{\prime}\]

where NT is replace by NT-k if debiased is True.
